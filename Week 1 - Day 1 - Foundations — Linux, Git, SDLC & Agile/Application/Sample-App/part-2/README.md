# Part 2 — Git Workflow & Agile Demo

## What This Covers
- Git repository initialization
- Staging and committing changes
- Branching and merging
- Pull request workflow simulation
- Agile sprint board simulation (Scrum)

## How to Run

```bash
# Make executable (once)
chmod +x git-workflow-demo.sh

# Run the demo
bash git-workflow-demo.sh
```

## What You'll See
A complete simulation of a feature development workflow using Git — from creating a repo, working in feature branches, resolving conflicts, and merging — plus an Agile sprint board showing story point tracking.
