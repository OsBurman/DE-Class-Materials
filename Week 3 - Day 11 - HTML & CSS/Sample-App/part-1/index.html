1. Semantic HTML Tags

Semantic tags describe meaning, not just appearance. They improve accessibility and SEO.

What is Semantic HTML?

Tags like <article>, <section>, <nav>, <aside>, <main>, <header>, and <footer> give the browser and screen readers meaningful context — unlike a generic <div>.

Normal text <span> 1 bold italic <span> 2 more text.
