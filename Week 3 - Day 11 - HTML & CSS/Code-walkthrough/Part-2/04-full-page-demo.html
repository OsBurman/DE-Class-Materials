Section 3

Specificity & Cascade

Four rules target the same paragraphs. The one with highest specificity wins.

p selector → gray (0,0,0,1)

.spec-class → blue (0,0,1,0)

.spec-class.spec-double → purple (0,0,2,0)

#spec-id → green (0,1,0,0) — wins

Specificity score: inline style (1,0,0,0) > #id (0,1,0,0) > .class / [attr] / :pseudo (0,0,1,0) > element (0,0,0,1)

Section 4

CSS Box Model

Every element is a rectangle with four layers. Open DevTools → hover an element → see the orange/green/blue overlay.

MARGIN (transparent, pushes other elements)
BORDER (coloured, adds to size in content-box)
PADDING (inside border, same background as content)
CONTENT (text, images, children)
width × height of actual content area

box-sizing: border-box (our global reset) — padding and border are included in the width you set.

box-sizing: content-box (browser default) — width is content only; total = width + padding + border.
